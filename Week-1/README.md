## Files and Descriptions

- `assgn1.js`
  - Compares two numbers and prints which one is bigger.

- `assgn2.js`
  - Compares three numbers and prints which one is the biggest.

- `assgn3.js`
  - Finds the smallest value in a marks array using a loop.

- `assgn4.js`
  - Defines a function that receives three numbers and prints which one is biggest.

- `assgn5.js`
  - Works with an `employees` array of objects: inserts a new employee, removes an employee by name, and updates a mark value.

- `assgn6.js`
  - Defines a function that receives an array and returns the sum of its values.

- `assgn7.js`
  - Defines a search function that returns the index of a target element in an array or returns `"not found"` if it does not exist.

- `assn8.js`
  - Calculates the sum of a marks array and prints the result.
