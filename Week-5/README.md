# Week 5: HTML/CSS Practice and React App Projects

## Overview
This week contains HTML/CSS practice pages and React starter apps created with Vite. The practice files focus on page layouts, tables, city guides, employee details, and responsive page structure.

This week focused on strengthening frontend development skills through:

Responsive Web Design
Flexbox Layouts
Tailwind CSS
Modern UI Development
React Fundamentals
Component-Based Architecture
The projects were designed to improve practical understanding of building responsive and interactive web applications using modern frontend technologies.

## How to View and Run
- HTML/CSS pages: open the `.html` files directly in a browser.
- React apps: open each app folder in a terminal and run:
  ```bash
  npm install
  npm run dev
  ```

## Notes
- The HTML/CSS folders are standalone practice examples.
- The React apps are Vite-based starter projects for frontend development practice.
