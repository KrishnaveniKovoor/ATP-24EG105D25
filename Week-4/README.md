# Week 4: HTML Practice and Web Page Layouts

## Overview
This week contains HTML practice files focused on foundational web page structure, headings, and simple layout work.

## Files Included
- `handson.html` — main HTML practice page.
- `handon2.html` — secondary HTML practice page.
- `h2.html` — heading and basic HTML structure practice.

## What’s Covered
- HTML document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Text elements and headings
- Paragraphs, links, and simple content layout
- Basic page formatting and design practice
